# World Cup Dashboard

## Project Overview
This project is a Power BI dashboard created to analyze FIFA World Cup data.

The dashboard provides interactive visualizations and insights related to:
- Team performance
- Match statistics
- Goals analysis
- Player performance
- Tournament trends

---

## Tools & Technologies
- Power BI
- Excel / CSV
- SQL
- GitHub

---

## Features
- Interactive dashboard
- KPI cards
- Team comparison
- Match analysis
- Dynamic filters and slicers
- Visual storytelling
